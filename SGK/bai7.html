Bài 7: Soạn thảo văn bản bằng tiếng Việt

CHỦ ĐỀ E: ỨNG DỤNG TIN HỌC

BÀI 7: SOẠN THẢO VĂN BẢN TIẾNG VIỆT

Mục tiêu: Biết cách gõ dấu tiếng Việt và lưu văn bản.

1. Kích hoạt Microsoft Word

Để mở phần mềm soạn thảo văn bản, em thực hiện:

Nháy đúp chuột vào biểu tượng Word trên màn hình nền.

Hoặc gõ "Word" vào ô tìm kiếm rồi nhấn Enter.

(Con trỏ soạn thảo là vạch đứng nhấp nháy | )

2. Quy tắc gõ Telex

Để gõ tiếng Việt, em cần bật Unikey (Chế độ chữ V màu đỏ).

Gõ Dấu
Sắc ( ' ) S
Huyền ( ` ) F
Hỏi ( ? ) R
Ngã ( ~ ) X
Nặng ( . ) J
Gõ Chữ
â aa
ô oo
ê ee
đ dd
ă aw
ơ ow

3. Lưu văn bản và Thoát

Lưu văn bản:

1. Chọn File -> Save (hoặc ấn Ctrl + S).

2. Chọn thư mục lưu (Browse).

3. Đặt tên tệp -> Nhấn Save.

Thoát:

Nháy chuột vào nút Close (X) ở góc trên bên phải màn hình.

Hướng dẫn học tập: Em hãy bấm mũi tên Trái/Phải trên khung hình trên để xem tóm tắt nội dung bài học.
