BÀI 9: BÀI TRÌNH CHIẾU CỦA EM

Mục tiêu: Tạo bài trình chiếu đơn giản, biết cách lưu và thoát phần mềm.

1. Khởi động và Thoát PowerPoint

Khởi động
Nháy đúp chuột vào biểu tượng trên màn hình.

Thoát
Nhấn nút Close (X) hoặc phím Alt + F4.

2. Các loại trang chiếu

Một bài trình chiếu thường gồm:

1. Trang tiêu đề
(Trang đầu tiên)
2. Trang nội dung
(Các trang tiếp theo)

3. Các thao tác cơ bản

  • Thêm trang mới: Thẻ Home -> New Slide.
  • Đổi dấu đầu dòng: Thẻ Home -> Bullets.
  • Lưu bài: File -> Save (hoặc Ctrl + S) -> Đặt tên -> Save.

Bấm mũi tên Trái/Phải trên khung hình để xem bài giảng.
